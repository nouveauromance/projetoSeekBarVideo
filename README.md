# Projeto SeekBarVideo

- Aplicativo mobile desenvolvido como atividade da disciplina de Programação para Dispositivos Móveis.

## Sobre o projeto

- O Projeto SeekBarVideo é um aplicativo mobile que permite que o usuário altere e escolha os vídeos exibidos no sistema.

## Escopo

- O aplicativo possui a seguinte funcionalidade:

  - Botão para alterar vídeo:
    - Usuário escolhe qual vídeo será exibido ao invés de mudar diretamente no código.

## Instalação

- Instale o projeto com npm:

```bash
  npm install projetoSeekBarVideo
  cd projetoSeekBarVideo
  npm start
```
